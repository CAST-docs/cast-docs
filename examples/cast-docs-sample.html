CAST Docs 模块化文档样例

一个克制、可离线打开、无外部依赖的 self-contained HTML 示例。

  • 输出形态：单个完整 HTML 文件，包含 inline CSS 和受控 inline JS。
  • 内容边界：用户内容不写脚本，交互能力由 renderer-owned modules 提供。
  • 视觉方向：借鉴 Lark/Docmost/Jsonita 的克制文档感，而不是复制具体品牌样式。

01整体定位

CAST Docs 的核心不是生成漂亮页面，而是把结构化技术文档稳定地渲染成可分享、可审查、可离线打开的 HTML。

INFO

模型只需要输出 JSON 和 manifest；模板、样式、组件 HTML、交互脚本都由 renderer 接管。

02模块架构

层职责配置来源
Document Schema定义 metadata、manifest、sections、typed blocksschemas/doc.schema.json
Block Registry把 block type 映射到 renderer、class、interactionconfig/components.json
Theme Resolver把主题 token 编译成 CSS variablesconfig/theme-tokens.json
Layout Shell选择 single-doc 或 document-set 壳config/layouts.json
Interaction Injector按需注入图表、复制、Finder 等交互模块config/interactions.json

03组件抽象

BlockSpec

每个块拥有 schema、renderer、allowed children、classes、validation 和 examples。

Renderer Hub

通过 block.type 分发渲染函数，避免按场景写硬编码分支。

Layout Shell

正文块不关心 sidebar、topbar、pagination，这些由 shell 统一提供。

Validator

分层检查 schema、manifest、HTML profile、交互 hook 和 unresolved placeholders。
