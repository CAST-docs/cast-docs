Summary

CAST Docs is a config-driven renderer for stable, self-contained technical documents.

Context

The renderer should produce consistent documents without copying large templates into model context.

Principles Or Features

Consistent outputUse one shared shell and semantic classes.
On-demand compositionRender only selected components.
Scenario skeletonsUse configured section order.
Reuse firstPrefer existing components and schema fragments.

Examples

python3 scripts/render_html.py --input doc.json --output doc.html --validate
